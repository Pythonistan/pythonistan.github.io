Python Felsefesi (Zen of Python)

13/Aug 2016 3 min. read

python prensip

Bugün teknik konulara biraz ara verip sizlere Python’un kullanımı ve kodlaması ile ilgili genel prensiplerden bahsetmek istiyorum. Python Felsefesi (The Zen of Python) Tercümesini vereceğim aforizmalar BDFL‘nin Python’u geliştirirken dikkate aldığı ana prensipleri özetlemek adına Python camiasına uzun yıllar emek vermiş bir Pythonista (Paytoncu) olan Tim Peters tarafından ortaya konmuş. Toplam sayısı 20 olan aforizmaların halihazırda 19‘u bilinmekte. Python Geliştirme Prensipleri yada Aforizmalar Buradaki prensip yada aforizmalara “Zen of Python” deniyor.

Notepad Kullanarak Python Programı Yazalım

5/Aug 2016 4 min. read

kodlama programlama python

Herhangi Bir Metin Editörü Evvelsi günkü blog yazımda bugün için Sublime Text (Sablaym Tekst) ile bir kod yazacağımı söylemiştim. Sublime’ı kurmadan önce kısa bir bilgi vermek istiyorum: Aslında Sablaym’a hiç ihtiyacınız yok! :) Programlarınızı herhangi bir metin editörü (notepad, wordpad, hatta ms word, v.d.) ile yazıp .py uzantısı ile kaydederek komut satırında aşağıdaki şekilde basitçe çalıştırabilirsiniz: python program_adi.py .py uzantısı bize bu dosyanın Python komut dosyası olduğunu belirtmektedir. Önemli: Programın çalışabilmesi için; Python sisteminizde kurulu olmalı ve programınız o anda komut satırının çalıştığı (yani imlecin yanıp söndüğü) dizinin içinde olmalıdır.

Python: Kurucusu - Tarihi - Okunuşu

3/Aug 2016 1 min. read

python tarih

Python’un İlk Geliştiricisi ve Kısa Tarihi Python programlama dilinin ilk geliştiricisi Hollandalı Guido van Rossum’dur. Kendisi Python camiasında “Ebedi Müşfik Diktatör” (Benevolent Dictator For Life-BDFL) olarak bilinir. Rossum Python hakkında şöyle diyor: Python’u 1989’da Unix/C hackerlarının hoşuna gidecek ve ABC dilinin izlerini taşıyacak bir hobi projesi olarak tasarladım. 2000’li yıllardan itibaren internetin hızlı gelişimi ve açık kaynak kodlu yazılımlara yazılımcıların verdiği büyük destekle Python çok geniş bir geliştirici ve kullanıcı tabanına sahip olmuştur.

Windows'ta Python Kurulumu

3/Aug 2015 2 min. read

python kurulum program

Bir önceki blog yazımda belirttiğim gibi Python’u bilgisayarınıza kurmadan Payton kodlarını bazı internet sitelerinde çalıştırabilirsiniz. Programlama yolculuğunun bir noktasında bu yaklaşım yetersiz hale geleceğinden ben şimdiden Payton’u kurmanızı tavsiye ediyorum. Şimdi Python (2.7 versiyonunu) bilgisayarımıza kuralım. Cevap olarak Python yazısı ve versiyonu çıkıyorsa (Foto 4) Payton kurulu demektir.

Peki Nedir Python

2/Aug 2015 2 min. read

algoritma programlama Python

Python Hakkında Python bir programlama dilidir. Programlama, Bilgisayar Biliminin (Computer Science) temelidir. Programlama dili ise Wikipedia‘ya göre, yazılımcının bir algoritmayı ifade etmek amacıyla, bir bilgisayara ne yapmasını istediğini anlatmasının tektipleştirilmiş yoludur. Programlama Dili Yani basitçe bilgisayara istediğimizi yaptırmak için ilettiğimiz komutların dilidir diyebiliriz. Yüksek düzeyli bir dil olan Python’un komutları İngilizce diline oldukça yakındır. Dilin düzeyi yükseldikçe komutlar doğal dillere (genelde İngilizce) yakın duruma gelmektedir, tersi durumda ise makina diline yakın hale gelmektedir.
