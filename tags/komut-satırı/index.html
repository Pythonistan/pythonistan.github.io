Windows'ta Python Kurulumu

3/Aug 2015 2 min. read

python kurulum program

Bir önceki blog yazımda belirttiğim gibi Python’u bilgisayarınıza kurmadan Payton kodlarını bazı internet sitelerinde çalıştırabilirsiniz. Programlama yolculuğunun bir noktasında bu yaklaşım yetersiz hale geleceğinden ben şimdiden Payton’u kurmanızı tavsiye ediyorum. Şimdi Python (2.7 versiyonunu) bilgisayarımıza kuralım. Sisteminizin Windows olduğunu varsayıyorum. Macintosh ve Linux sistemlerin çoğunda Python ön kurulu olarak gelmektedir. Sisteminizde Payton’un kurulu olup olmadığını anlamak için komut satırı (terminal olarak da geçer) programınızı açıp python komutunu yazmanız yeterli. Cevap olarak Python yazısı ve versiyonu çıkıyorsa (Foto 4) Payton kurulu demektir.
