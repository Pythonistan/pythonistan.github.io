Git Nedir? Nerede ve Nasıl Kullanılır?

21/Aug 2016 6 min. read

GIT GITHUB WEBSİTE KURULUM

Git ile Tanışma Bir önceki yazımda bahsettiğim Hugo ve Jekyll ile web sitesi yapma olayına girişmeden önce üzerinde durmamız gereken bir konu var: Git. Bazı temel git komutlarını öğrenmemiz gerekiyor zira web sitemizi oluştururken dosya indirme ve siteyi oluşturduktan sonra ise sitemizi web server’a yüklemek için bu komutları kullanacağız. Git bizim için sadece dosyalarımızı yedeklemek yada güncellemeleri tutmakla kalmayıp dosyaları web’e transfer etmede kullanacağımız bir yol olacak. Eskiden bu iş için ftp protokolu ve ftp programları kullanılırdı.
