Python: Kurucusu - Tarihi - Okunuşu

3/Aug 2016 1 min. read

python tarih

Python’un İlk Geliştiricisi ve Kısa Tarihi Python programlama dilinin ilk geliştiricisi Hollandalı Guido van Rossum’dur. Kendisi Python camiasında “Ebedi Müşfik Diktatör” (Benevolent Dictator For Life-BDFL) olarak bilinir. Rossum Python hakkında şöyle diyor: Python’u 1989’da Unix/C hackerlarının hoşuna gidecek ve ABC dilinin izlerini taşıyacak bir hobi projesi olarak tasarladım. 2000’li yıllardan itibaren internetin hızlı gelişimi ve açık kaynak kodlu yazılımlara yazılımcıların verdiği büyük destekle Python çok geniş bir geliştirici ve kullanıcı tabanına sahip olmuştur.
