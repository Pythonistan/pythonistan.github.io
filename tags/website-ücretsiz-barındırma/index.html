HUGO GIT WEBSİTE GITHUB HOSTING

Nihayet bugün, Hugo ile statik bir websitesi nasıl yapılır serimizi beşinci bölümü ile sonlandırıyoruz. Birinci bölümde Hugo kurulumunu, ikinci bölümde Hugo ile Siteyi Lokal Bilgisayarda Oluşturma‘yı, üçüncü bölümde Temamızı Özelleştirme‘yi, dördüncü bölümde ise post sayfalarını editleme konusunu işlemiştik. Bugünkü konumuz, sitemizi ücretsiz olarak Github‘da barındırma ve bu işlemi git vasıtası ile nasıl yapacağımız.
