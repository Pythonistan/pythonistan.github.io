Python’un İlk Geliştiricisi ve Kısa Tarihi

Python programlama dilinin ilk geliştiricisi Hollandalı Guido van Rossum’dur. Kendisi Python camiasında “Ebedi Müşfik Diktatör” (Benevolent Dictator For Life-BDFL) olarak bilinir. Rossum Python hakkında şöyle diyor:

Python’u 1989’da Unix/C hackerlarının hoşuna gidecek ve ABC dilinin izlerini taşıyacak bir hobi projesi olarak tasarladım.

2000’li yıllardan itibaren internetin hızlı gelişimi ve açık kaynak kodlu yazılımlara yazılımcıların verdiği büyük destekle Python çok geniş bir geliştirici ve kullanıcı tabanına sahip olmuştur.

Python İsminin Ortaya Çıkışı

İnternette bazı Python sitelerinde piton yılanı simgesi görmüş olabilirsiniz. Sanılanın aksine bu programlama dilinin ismi bir yılan cinsi olan Piton’dan değil, Rossum’un çok sevdiği bir İngiliz radyo tiyatrosu oyunu “Monty Python’s Flying Circus”‘dan gelmektedir.

Python Kelimesinin Muhtelif Okunuşları

Türkiye’de Python programlama diline orijinal okunuşuna benzer şekilde “Paytın” denildiği gibi belli bir kitle Türkçede kullanılan yılan ismi “Piton”u kullanıyor. Payton kullanımı zaten mevcut. Blogumda Türkçe’nin fonetiğine en fazla uyduğunu düşündüğüm “Payton” kullanımını tercih ediyorum.
