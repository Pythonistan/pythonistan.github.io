Windows'ta Python Kurulumu

3/Aug 2015
PYTHON KURULUM PROGRAM KOMUT SATIRI
Like Google +1 Tweet

Bir önceki blog yazımda belirttiğim gibi Python’u bilgisayarınıza kurmadan Payton kodlarını bazı internet sitelerinde çalıştırabilirsiniz. Programlama yolculuğunun bir noktasında bu yaklaşım yetersiz hale geleceğinden ben şimdiden Payton’u kurmanızı tavsiye ediyorum.

Şimdi Python (2.7 versiyonunu) bilgisayarımıza kuralım. Sisteminizin Windows olduğunu varsayıyorum. Macintosh ve Linux sistemlerin çoğunda Python ön kurulu olarak gelmektedir. Sisteminizde Payton’un kurulu olup olmadığını anlamak için komut satırı (terminal olarak da geçer) programınızı açıp python komutunu yazmanız yeterli. Cevap olarak Python yazısı ve versiyonu çıkıyorsa (Foto 4) Payton kurulu demektir. Mesela Python 2.7.11 gibi.

Ben sitede Payton’un 2.x versiyonunu kullanacağım. Zira Payton 3 versiyonunda bariz söz dizimi(syntax) değişiklikleri var, ayrıca bu versiyon henüz tam olarak yaygınlaşmadı. Dolayısıyla 2.x versiyonunu indirmenizi tavsiye ediyorum.

Python 2.7

  1. Python.org sitesine gidin -> Python Org ve Downloads kısmına girin. (Foto 1)
  2. Python 2.7.x versiyonunu _(yada yeni bir 2.x versiyonu çıktıysa onu) bilgisayarınıza indirin. (Foto 2)
  3. İndirdiğiniz dosyayı çift tıklayarak kurmaya başlayın.
  4. Kurulumda çeşitli ekranlar gelecek. Bu ekranları Next diyerek geçebilirsiniz. Ancak Python’u ortam değişkenlerine kaydedeceğimiz ekranı hızlı geçmeyin, Python’u Path(Ortam) değişkenine eklemeyi unutmayın. Sizdeki ekran farklı olabilir, PATH kelimesine dikkat edin.(Foto 3)
  5. Sorunsuz şekilde kurulum gerçekleşmişse komut satırında (Windows’ta, Başlat-Çalıştır-Cmd yada Başlat-Windows PowerShell, Mac ve Linux’ta Terminal) python yada python -V (Payton versiyonunu öğrenmek için kullanılır) komutlarına hatasız cevap almanız gerekir. (Foto 4)

Şimdi artık bilgisayarımızda Python‘u kullanabiliriz. Bir sonraki blog postumda sizinle beraber komut satırından çok basit bir program yazacağız. O zamana kadar hoşçakalın.
