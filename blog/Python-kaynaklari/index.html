Python Kaynakları

Bloguma şöyle bir baktım da Python ile ilgili son yazıyı yayımlayalı bayağı zaman geçmiş. Bu yazıda Python hakkında yararlandığım kaynaklardan bir seçki yapmaya karar verdim.

Python popülerlik sıralamasında hemen her programlama listesinde ilk 5’e girdiğinden kendisiyle ilgili internette yüzbinlerce kaynak bulmak mümkün.

Online olarak okuyabileceğiniz kitaplardan, alabileceğiniz online kurslara, profesyonel Pythoncuların bloglarından sadece Python’a özgü websitelerine kadar binbir çeşit kaynak var.

Tek başına StackOverFlow sitesi bile devasa bir kaynak teşkil ediyor. Stackoverflow‘a ayrı bir yazımda detaylı değinmek istiyorum.

Python kaynaklarının hepsine birden yer vermem imkansız. Ben en çok faydalandığım / kullandığım kaynakları aşağıda vereceğim.

Sizin faydalandığınız güzel kaynaklar varsa yorum kısmından bilgilendirip bu listede yer almasını sağlayabilirsiniz.

Türkçe Kaynaklar

Öncelikle Türkçe kaynaklara bir göz atalım. Bu dalda İngilizce’de olduğu kadar bir çeşitlilik ve miktardan söz edemiyoruz. Ancak çok kaliteli kaynakların varlığını göz ardı etmemek lazım.

  1. Türkçe Python kaynak denince Yazar Fırat Özgül‘ün Python orjinal klavuzlarını Türkçeye çevirdiği Python kitapları akla geliyor. Motamot çeviri olmadığı için kitaplar oldukça başarılı. Her Paytoncu’nun yanından ayırmaması gereken bir başucu kitabı.

    Kitaplar Pyton 3 ve Python 2 olarak istihza.com adresinde online olarak kategorilere ayrılmış olarak sunuluyor. Siteden kitaptaki örnekleri kontrol edebilirsiniz. İstihza websitesinin forum kısmı Türkçe bir siteye göre nisbeten aktif sayılabilir.

    Kitapları ayrıca bilgisayarınıza pdf olarak indirebilir, cep telefonunuzda mobil kitap okuyucu (epub reader) ile açıp okuyabilir ve içindeki güzel örneklerden gerçekten çok faydalanabilirsiniz. Üzerinde ciddi emek sarf edilmiş bir eser olduğu rahatlıkla söylenebilir.

  2. Pythondersleri değişik Python’cuların Python hakkında blog yazılarını yayımladıkları bir site. Çok sık güncellenmese de dikkate değer bir site diyebilirim.

  3. DjangoGirls, Python temelli web framework Django‘yu kadınlara sevdirmek ve Django‘nun kullanımını yaygınlaştırmaya çalışan kar amacı gütmeyen bir organizasyon.

    ABD kaynaklı bu organizasyonunun Django‘yu anlatan klavuzunun Türkçe çevirisi Python ile web geliştirme yapacak olanlar için kullanılabilir bir kaynak.

İngilizce Kaynaklar

İngilizce kaynaklar hem çeşitlilik hem de hacim olarak Türkçe kaynaklara göre doğal olarak çok daha geniş. Şimdi kategoriler halinde inceleyelim.

Ücretsiz Python Kitapları

  1. Ücretsiz Python kitapları için aşağıdaki sayfadan ulaşabileceğiniz linklerden faydalanabilirsiniz: FreeComputerBooks / Python Bu sayfada 50 - 60 adet güzel kitaba referans var. O’relly gibi kaliteli içerik üreten yayınevi kitaplarına ulaşabilirsiniz. Kitapları okuyabileceğiniz linkler her kitabın kendi sayfasında Read and Download Links: kısmının altında bulunuyor.

  2. Wikibooks sitesinde Python üzerine yazılmış özellikle başlangıç seviyede bir çok kitaba erişmeniz mümkün. Bakmanız gereken adres: Python Wikibooks .

  3. Tecrübeli programcı Zed Shaw tarafından yazılan ve içeriği itibarı ile “farklı ve sıradışı” bir öğretim stratejisi güden Learn Python the Hard Way kitabına ücretsiz olarak ulaşabilirsiniz.

Python Websiteleri

  1. Python websitelerine Python.org ile başlamamak olmaz heralde. Özellikle dokümantasyon bölümünde binlerce sayfalık kaynak bulacaksınız.

  2. Python Central sitesi Python ipuçları, klavuzları , kütüphaneleri , araçları bölümlerini bulabileceğiniz zengin bir kaynak.

  3. Python Programming Python‘un gelişmiş kullanımları ile ilgili klavuzlar ve nasıl yapılır dokümanları sağlıyor. Machine Learning, Data Analysis, Robotics, v.d. konularda bir kısmı video açıklamalı çok yararlı bilgilere ulaşabilirsiniz.

  4. Python ile web geliştirme ve diğer ayrıntılı konuları açık ve sade bir dille anlatmayı hedefleyen Full Stack Python sitesi açık bir kitaba (open book) ev sahipliği yapıyor. Özellikle başlangıç seviyesinden orta seviyeye atlamak için faydalı bir kaynak.

Python Interaktif Websiteleri

  1. Learn Python websitesinde hem Python genel konseptlerine aşinalık sağlayabilir hem de verilen kod örneklerini interaktif olarak deneyerek görebilirsiniz.

  2. İlk yazılarımdan birinde Python Tutor sitesinden bahsetmiştim. Bu sitede yazdığınız kod parçacıklarını interaktif olarak deneyebilir ve kodların interaktif haritasını çıkarabilirsiniz.

  3. Interactive Python sitesinde bir çok interaktif Python kitabına ulaşabilir ve test edebilirsiniz. Ücretsiz olarak üye olup içeriğe ulaşabilirsiniz.

  4. Bazı kod bloklarının nasıl çalıştığını görmek için Intro to Python sitesine göz atabilirsiniz.

Python Üzerine Online Kurs ve Akademiler

  1. Code Academy interaktif ve adım adım Python kursları ve dersler sunuyor.

  2. Codementor sitesi değişik seviyeler için Python kurslarına sahip. Üyelik ücretsiz.

  3. Udacity giriş ve orta seviyeli tüm bilgisayar bilimleri (CS) kurslarında Python kullanmakta. Yüzbin insanın aldığı Bilgisayar Bilimine Giriş (Intro to Computer Science) kursu Python ile işlenmiş, sitede daha üst seviyeli Python kursları da mevcut. Üyelik ücretsiz. Dersleri videolardan takip ediyorsunuz. Video aralarında kısa testlerle anlamanızı pekiştiriyorsunuz.

  4. Coursera muhtemelen internetin en bilinen MOOC‘si. (bu kavram çok fazla sayıda kişinin iştirak ettiği herkese açık internet kurslarını ifade etmek için kullanılıyor - Massive Open Online Course) Site çok sayıda Python içerikli kursa sahip. İçerik yüksek kalite olarak değerlendiriliyor.

  5. EdX Harvard, MIT, Berkeley gibi ABD’nin saygın birkaç üniversitesinin bir araya gelerek oluşturduğu bir MOOC sitesi. Coursera gibi burada da yüksek kalitede içerik bulabilirsiniz. İçerikler ücretsiz, ancak sertifika almak isterseniz belli bir ücret karşılığında alabilirsiniz.

Python Blogcuları

  1. Programcı Trey Hunner‘ın blogu Python üzerine ilgi çekici yazılar barındırıyor.

  2. Daha tecrübeli Paytoncular için Doug Hellman‘ın blogu yardımcı olabilir.

  3. Python hakkındaki son haberleri ve Python geliştiricilerini tanıyabileceğiniz bir blog Python Library adresinde yer alıyor.

  4. Planet Python sitesinde bir çok Paytoncunun bloguna tek bir sayfadan ulaşma imkanına sahip olacaksınız.

  5. Sonuncu olarak yer vereceğimiz ancak önem olarak diğerlerinden hiç de aşağıda kalmayan Python Weekly websitesini favorilerinize alın ve bu haftalık online dergiye üye olun. Python ile ilgili en son haberlere erişeceksiniz.

Burada saydığım kaynaklardan çok daha fazlasını Python üzerine yapacağınız basit bir Google araması ile bulabilirsiniz.

Sizlere Python ile iyi ve verimli çalışmalar dilerim…
